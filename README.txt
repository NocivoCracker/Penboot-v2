===========================[Penboot project:

execute o arquivo dependencias.sh.x para instalar as dependencias necessárias do penboot v2.
o arquivo penboot2.py deverá ser executado com o python na versão 3.x.

como executar o arquivo dependencias.sh.x :
Dê a permissão de execução: chmod +x dependencias.sh.x
execute o arquivo: ./dependencias.sh.x

como executar o arquivo penboot2.pyc:
Dê a permissão de execuão: chmod +x penboot.pyc
execute o arquivo: ./penboot.pyc

#Novidades da atualização(BETA):
*Interface gráfica interativa.

Obs: Este programa funciona apenas em distribuições Linux.
